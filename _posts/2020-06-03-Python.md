## staticmethod

functions with @staticmethod don't need to be substantialized
```
class ex:
	@staticmethod
	def a():
		print('m')

ex.a()
# output: m
```

## classmethod

functions with @classmethod can use attributes and functions in the same class without substantiation.

```
class ex:
	def a():
		print('a')
	@classmethod
	def b(cls):
		cls.a()

m = ex()
m.b()

# output: a
```
